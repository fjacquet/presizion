<objective>
Improve SPECrate workflow by auto-deriving socket/core fields from benchmark metadata, making them read-only in specint mode, and adding a lookup link that copies the CPU model to clipboard before opening the SPEC results page.

Purpose: Presales engineers importing LiveOptics data should not need to manually re-enter socket/core counts that are already parsed from the import. The lookup link copies the CPU model to clipboard so the user can paste it directly into the SPEC search page (SPEC CGI does not support URL pre-fill).

Output: ScenarioCard fields become read-only when metadata is available in specint mode; CurrentClusterForm gains a conditional "Look up SPECrate" button that copies CPU model to clipboard and opens SPEC results page.
</objective>

   async function handleSpecLookup() {
