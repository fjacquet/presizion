**Step 1: Configure path aliases (required BEFORE shadcn init).**

In `tsconfig.app.json`, inside `compilerOptions`, add:

```json
"baseUrl": ".",
"paths": { "@/*": ["./src/*"] }
```

In `vite.config.ts`, add the path resolver. Import `path` from `'path'` and add inside `defineConfig`:

```typescript
import path from 'path'

// inside defineConfig:
resolve: {
  alias: { '@': path.resolve(__dirname, './src') },
},
css: {
  postcss: {
    plugins: [tailwindcss()],
  },
},
```

Also add at the top of `vite.config.ts`:

```typescript
import tailwindcss from '@tailwindcss/postcss'
```
